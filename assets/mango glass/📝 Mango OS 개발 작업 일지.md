📝 Mango OS 개발 작업 일지

1. 프로젝트 현황
   날짜: 2026년 3월 14일

작업자: 대표/메인 디자이너 권한결 (@KHANKYUL)

주요 성과: Pi Network 메인넷 실결제 엔진 연동 성공 및 생태계 확장 UI 구축

2. 금일 작업 상세 (Technical Summary)
   ① 권한 시스템 동기화 (Auth Sync)
   내용: 관리자 권한 명칭을 TOP_SUPER_ADMIN으로 단일화.

교정: authPi 함수에서 호출하는 명칭과 renderMenuByRole 필터링 함수의 매개변수를 일치시켜 대표님 접속 시 관리자 제어판이 즉시 개방되도록 수정.

② 생태계 참여 UI 배치 (UI/UX)
위치: 프로필 카드 내 유저 ID(@KHANKYUL) 바로 하단 (약 136행).

추가 요소: \* 활동가(REPORTER) 신청 버튼: 부산 지역 맛집 및 정보 수집가 모집용.

가맹점(PARTNER) 등록 신청 버튼: 실제 Pi 결제 매장 확보 및 비즈니스 확장용.

디자인: Mango OS 고유의 브라운 골드 테마 및 backdrop-blur 유리 질감 유지.

③ 하이브리드 가입 로직 설계 (Compliance)
원칙: Pi Network 보안 가이드라인에 따라 별도 ID/PW 없이 **Pi Auth(SDK)**만 사용.

구조: applyForRole() 함수를 통해 유저의 고유 UID와 신청 직무, 지역(Busan), 타임스탬프를 DB로 전송하는 기초 설계 완료.

3. 향후 5년 유지 및 확장 계획
   데이터 보존: 모든 활동가 및 가맹점 신청 내역은 5년 이상 영구 보존 가능한 DB 구조 채택.

실사용 증명 (Step 10): 부산 지역 서포터즈 5인 이상을 통해 실결제 트래픽 및 데이터 생성 유도.

승인 프로세스: 모든 등업 신청은 CEO(@KHANKYUL)의 최종 승인을 거쳐야만 반영되도록 보안 강화.
