로컬 개발 환경(VS Code)에서 이 코드를 실행하기 위한 설정 방법과 파일 구조를 안내해 드립니다.

이 코드는 React와 Tailwind CSS를 기반으로 작성되었습니다. 가장 빠르고 표준적인 방법인 Vite를 사용하는 것을 추천합니다.

1. 프로젝트 생성 및 위치
터미널(또는 CMD)을 열고 프로젝트를 만들고 싶은 폴더로 이동한 뒤 아래 명령어를 입력하세요.

Bash

# 프로젝트 생성 (프로젝트명: mango-travel)
npm create vite@latest mango-travel -- --template react

# 폴더 이동
cd mango-travel

# 필요한 패키지 설치
npm install
npm install lucide-react firebase
npm install -D tailwindcss border-left postcss autoprefixer
npx tailwindcss init -p
2. 파일명 및 저장 위치
VS Code에서 mango-travel 폴더를 엽니다.

파일명: App.jsx

저장 위치: src/App.jsx (기존에 있는 내용을 모두 지우고 제가 드린 코드를 복사해서 붙여넣으세요.)

3. 추가 설정 (Tailwind CSS)
Tailwind가 정상 작동하려면 tailwind.config.js 파일을 아래와 같이 수정해야 합니다.

4. 실행 방법
터미널에서 다음 명령어를 입력하면 로컬 서버가 실행됩니다.

Bash

npm run dev
터미널에 뜨는 주소(보통 `http://localhost:5173`)를 컨트롤(Ctrl) 키를 누른 채 클릭하면 브라우저에서 확인할 수 있습니다.

### 주의사항
* **Firebase 설정:** 코드 상단에 있는 `__firebase_config`와 `__app_id` 등은 현재 이 브라우저 환경에서 제공하는 변수입니다. 로컬에서 실행할 때는 본인의 Firebase 콘솔에서 발급받은 실제 설정값 객체로 교체해야 데이터 저장이 정상 작동합니다.
* **이미지:** Unsplash 이미지 링크를 사용하고 있으므로 인터넷이 연결된 상태여야 이미지가 보입니다.
