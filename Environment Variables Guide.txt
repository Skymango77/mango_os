1. 파일 만들기

프로젝트 폴더 최상단에 .env라는 이름의 파일을 만듭니다. (파일 확장자가 따로 없습니다.)

2. 내용 작성법

파일 안에 아래와 같이 변수명과 값을 적습니다. (따옴표는 보통 쓰지 않습니다.)

# 백엔드 서버용 (Pi API Key)
PI_API_KEY=your_copied_api_key_here

# 프론트엔드에서 혹시 필요할 경우 (React 기준)
REACT_APP_PI_APP_ID=your_app_id


3. 보안 규칙 (매우 중요!)

.gitignore에 추가: .env 파일이 GitHub 같은 곳에 업로드되지 않도록 반드시 .gitignore 파일에 .env 한 줄을 추가해야 합니다.

백엔드 전용: PI_API_KEY는 나중에 Node.js나 Python 같은 서버 코드에서만 불러와서 사용할 것입니다.

4. 다음 단계 진행

이제 API Key 설정은 잊으셔도 됩니다. 다음 단계로 사용자가 앱에 접속했을 때 Pi 계정으로 로그인하고 유저 정보를 화면에 띄우는 기능을 구현해 보겠습니다.
