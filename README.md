# 모달 테두리 5배 강화 및 울트라 네온 적용 완료

## 1. 수정 핵심 (Heavy-Duty Design)

- **Thickness Upgrade**: `border: 10px`를 적용하여 시각적으로 압도적인 프레임 두께 확보.
- **Triple Glow**: 테두리 선 자체에서 빛이 뿜어져 나오는 효과를 위해 3단계 중첩 `box-shadow` 설계.
- **Visual Impact**: 텍스트에 `font-[1000]`과 `drop-shadow`를 추가하여 두꺼운 테두리 사이에서도 밀리지 않는 가독성 확보.

## 2. 사용자 인터랙션

- **Pulse Extreme**: 테두리 색상과 빛의 세기가 2초마다 강렬하게 변하며 '중요 알림'임을 직관적으로 인지시킴.
- **Confirm Button**: 버튼 역시 두껍고 입체감 있게(border-2) 수정하여 전체적인 톤앤매너 일치.

# 테두리 40px 울트라 네온 스타일 검수 보고서

## 1. 보정 사항

- **Visual Balance**: `border: 40px`가 적용될 경우 모달 내부 `padding`을 최소 `p-12` 이상 주어야 글자가 테두리에 먹히지 않습니다.
- **Performance**: `box-shadow` 값이 너무 크면 모바일에서 버벅일 수 있어 `filter: brightness`를 섞어 효율적으로 발광 효과를 구현했습니다.
- **Rotation Fix**: `gold-dust-effect`의 애니메이션 범위를 `-100%`로 수정하여 회전 시 사각지대가 보이지 않도록 처리했습니다.

## 2. 권장 적용법

- 해당 스타일을 적용할 때 모달의 전체 너비를 `w-[95%]` 정도로 넓게 잡아야 40px의 두꺼운 프레임이 아름답게 표현됩니다.

# index.html CSS 검수 및 반사 효과 디버깅 완료

## 1. 발견된 문제점

- **Reflection Conflict**: `-webkit-box-reflect`가 3D 변환(`preserve-3d`) 요소에서 렌더링되지 않는 현상 확인.
- **Height Issue**: 반사된 잔상이 표시될 공간(Height)이 부족하여 화면 하단에 가려짐.
- **Duplicate Selectors**: `.glass` 클래스가 중복 정의되어 속성이 덮어씌워질 위험이 있음.

## 2. 해결 방법

- `.carousel-view`의 높이를 `250px` -> `320px`로 상향 조정.
- 반사 마스크의 투명도 시작점(Transparent)을 `40%` -> `50%`로 늦춰 실제 형체가 더 많이 반사되도록 변경.
- 바닥 글로우(`::after`)의 강도를 높여 입체감 부여.

## 3. 적용 가이드

- VS Code에서 기존 `.carousel-view`와 `.carousel-pivot` 스타일을 위 수정 코드로 덮어씌우세요.
- 만약 모바일 Chrome에서 여전히 안 보인다면, `box-reflect` 대신 가상의 `::before` 요소를 만들어 수동 반사를 구현해야 합니다.

# [2026-02-24] Mango Token & Pay 파일 시스템 구축

## 1. 파일 생성 정보

- **경로:** `mango_pay/core/`
- **파일명:** `token-logic.js`
- **목적:** Mango Token(MNG) 발행 로직 및 Pi 코인 연동 환율 제어

## 2. 코드 뼈대 (복사해서 token-logic.js에 저장)

```javascript
/**
 * Mango Token (MNG) Logic v1.0
 * Context: 2026-2029 Mango Ecosystem Strategy
 */

const MangoTokenSystem = {
  symbol: "MNG",
  baseRate: 1000, // 1 Pi = 1000 MNG

  // 유저의 망고 토큰 잔액 가져오기
  fetchBalance: function () {
    return localStorage.getItem("mng_balance") || "0.00";
  },

  // Pi 결제 성공 후 토큰 발행
  mintToken: function (piAmount) {
    const minted = piAmount * this.baseRate;
    let current = parseFloat(this.fetchBalance());
    localStorage.setItem("mng_balance", (current + minted).toFixed(2));
    return minted;
  },
};
```
