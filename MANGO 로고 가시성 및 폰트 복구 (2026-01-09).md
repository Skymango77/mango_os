# MANGO 로고 가시성 및 폰트 복구 (2026-01-09)

## 1. 문제 분석

- 폰트가 'Kanit'으로 설정되어 이전의 통통한 느낌이 사라짐.
- `text-fill-color: transparent` 속성으로 인해 글자 외곽선이 얇아 보이는 현상 발생.

## 2. 조치 사항

- **폰트 변경**: `Kanit` → `Fredoka One` (가장 통통하고 둥근 폰트).
- **두께 복구**: 글자를 투명하게 만드는 속성을 제거하여 폰트 본연의 두께감을 확보.
- **기능 유지**: 금가루 입자 효과, 맥동(Pulse) 애니메이션, 금빛 반짝임 효과는 원본 그대로 유지.

## 3. 코드 위치

- **HTML Header**: 약 12행 (폰트 링크)
- **CSS Style**: 약 72행 (`.mango-gold-power` 클래스)
